# 🌿 ROOFI – Eco Fashion Website

**ROOFI** là website thời trang thân thiện với môi trường, hướng đến phong cách **Minimal – Eco – Timeless**.  
Dự án được thực hiện như một bài tập học phần *Business Web Development (BIE5030)* tại Trường Đại học Kinh tế – Luật (UEL).

---

## 🖥️ Tổng quan dự án
Website giới thiệu và bán sản phẩm thời trang bền vững, kết hợp trải nghiệm mua sắm online hiện đại với thông điệp “**Be Kind to Earth, Be True to Style**”.

### 🔹 Các trang chính:
| Trang | Chức năng chính |
|-------|------------------|
| `index.html` | Trang chủ (banner, giới thiệu, sản phẩm nổi bật, reviews, contact form) |
| `product.html` | Danh mục sản phẩm (lọc theo style: Hobo, Streetwear, Casual, Minimal) |
| `product-detail.html` | Chi tiết sản phẩm (ảnh, giá, mô tả, chất liệu, size, đánh giá) |
| `cart.html` | Giỏ hàng, tính tổng tiền, form nhập thông tin đặt hàng |
| `order_confirm.html` | Trang xác nhận đơn hàng thành công |
| `page/qna.html` | Trang Câu hỏi thường gặp (FAQ / Accordion) |
| `page/hot-offers.html` | Trang Khuyến mãi nổi bật có đồng hồ đếm ngược |
| `page/about.html` | Giới thiệu thương hiệu ROOFI |
| `page/blog.html` | Blog chia sẻ phong cách sống xanh |
| `page/auth/login.html` | Trang đăng nhập người dùng |

---

## 🧠 Công nghệ sử dụng
- **HTML5 / CSS3 / JavaScript**
- **Remixicon** (biểu tượng giao diện)
- **Google Fonts** (Montserrat, Roboto)
- **Axios + jQuery** (xử lý API & DOM)
- **API hành chính Việt Nam** (điền tự động tỉnh – huyện – xã trong form đặt hàng)

---

## 🧩 Các tính năng nổi bật
- 🛒 Giỏ hàng động (cập nhật tổng giá, xóa, thay đổi số lượng)
- 💬 Accordion Q&A tương tác
- ⏰ Đồng hồ đếm ngược khuyến mãi (Hot Offer Timer)
- 📦 Form đặt hàng có tự động điền địa chỉ bằng API tỉnh/thành
- 📱 Giao diện **responsive** cho mobile
- 🧾 Form liên hệ & đăng ký nhận tin (Newsletter)
- 🌍 Kết nối đa trang: Blog – About – Hot Offers – Q&A – Login

## 🚀 Cách chạy project
1. Tải toàn bộ thư mục về máy  
2. Giải nén và mở file `index.html` trong trình duyệt  
3. Đảm bảo cấu trúc thư mục **asset/** không bị đổi chỗ  

> Nếu bạn muốn deploy lên GitHub Pages:  
> chỉ cần push toàn bộ lên nhánh `main` và bật **Pages → Deploy from branch**.
